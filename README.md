# Marvelous

Marvelous is a cross platform markdown editor and viewer built using Github Atom's shell - Electron.

## How To

### 1. clone this repo

``` bash
git clone https://github.com/hashd/Marvelous.git
```

### 2. `cd` into the repo directory and run `npm install`

``` bash
cd Marvelous
npm install
```

### 3. run `npm start`

``` bash
npm start
```
